# AIUI_Project

Tech Used:

Frontend

React 18 + Vite — component framework, dev server at localhost:5173
React Router DOM v6 — client-side routing
React Leaflet + Leaflet — interactive map with custom SVG pins
CartoDB Positron tiles — the light map style
react-datepicker — date range picker on Create Post
Axios — HTTP client for API calls
CSS modules per page — no CSS framework, all hand-written


Backend

Node.js + Express — REST API on port 5001
MongoDB Atlas — cloud database
Mongoose — ODM for schema/models
bcryptjs — password hashing
jsonwebtoken (JWT) — auth tokens (7-day expiry)
dotenv — environment variable management
